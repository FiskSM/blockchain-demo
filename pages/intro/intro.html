Blockchain Demo

A web-based demonstration of blockchain concepts.

This is a fork of Anders Brownworth's Blockchain Demo.
Watch his excellent Visual Demo here.

The following changes have been made in this fork:
  • Static HTML/JS, so it can be served with GitHub Pages
  • Use AngularJS for rendering the page
  • Show concept of mining difficulty
  • Show hashing speed
